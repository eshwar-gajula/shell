# Termux Pentester Shell

Professional custom shell for Android Termux.

## Features

- Clean startup banner
- Username system
- Time and date
- System info
- Git-aware prompt
- Backup support

## Installation

```bash
pkg install git -y
git clone https://github.com/eshwargajula31/termux-pentester-shell
cd termux-pentester-shell
chmod +x install.sh
./install.sh
